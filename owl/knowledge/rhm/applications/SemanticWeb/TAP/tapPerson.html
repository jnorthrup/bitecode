Notes

  • "x isa* y" means "x iss* y" if x is a concept.
    "x isa* y" means "x isu* y" if x is a unit.
  • The TAP KB contains a contradiction;
    it considers Person to be both a concept
    Person iss Agent;
    and a unit
    Person isu Class;

\\    Resource
ke$

ke$ Person isu* ?;
