Notes

  • "x isa y;" means "x iss y;" if x is a concept.
    "x isa y;" means "x isu y;" if x is a unit.
  • Person isa* ?;

  • The OpenCyc KB contains a contradiction;
    it considers Person to be both a unit
    Person isu Collection;
    and a concept
    Person iss Individual;
