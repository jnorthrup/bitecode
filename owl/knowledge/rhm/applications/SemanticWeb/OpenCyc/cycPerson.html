OpenCyc: Person isa* ?;

Notes

  • The bottom line
  • "x isa* y" means "x iss* y" if x is a concept.
    "x isa* y" means "x isu* y" if x is a unit.
  • "x isu* ?;" denotes x and all of its genus
    concepts, i.e.,
    x isu y1; y1 iss y2; ... yn iss Thing;
    The intermediate things, yi, are not units.
  • These are all contradictions; y is both a
    concept (collection) and a unit (individual).
    x isu**2 z
    x isu y; y isu z;
    x iss y; y isu z;
  • The OpenCyc KB contains many such contradictions
    (which would form infinite loops, if considered valid).
  • Some contradictions ignored
    x isu x;
    #$Thing isu x;
    #$Collection isu x;
    #$CollectionType isu x;
    #$CollectionTypeType isu x;
    #$AtemporalNecessarilyEssentialCollectionType isu x;
  • context
    #$UniversalVocabularyMt
  • #$Person iss* #$Thing lattice
    not simplified: 2840 different paths
    simplified: 1172 different paths
